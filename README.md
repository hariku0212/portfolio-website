## GitHub Pages の有効化手順

1. GitHub リポジトリの **Settings** → **Pages** を開く
2. **Source** を `Deploy from a branch` に設定
3. **Branch** を `main` / `/(root)` に設定して Save
4. 数分後に `https://hariku0212.github.io/portfolio-website/` で公開される
